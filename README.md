# Quote Generator

A simple Quote Generator built with HTML, CSS, and JavaScript that fetches random quotes from an API and allows users to share them on Twitter.

## Live Demo
ðŸ”— https://asma-ehsan.github.io/Quote--Generator/

## Features
- Generates a random quote on button click
- Displays quote text and author
- Fetches data from an external API
- Tweet the quote directly on Twitter
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API
- Font Awesome

## How It Works
- The app fetches a random quote from the API.
- The quote text and author are displayed on the screen.
- Clicking **New Quote** loads a fresh quote.
- Clicking **Tweet** opens Twitter with the quote ready to post.

## API Used
```
https://dummyjson.com/quotes/random
```
